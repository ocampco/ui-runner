# :runner: ui-runner

Simple ui runner using [create-react-app](https://github.com/facebook/create-react-app).

### Commands

```sh
# Install dependencies
$ nvm use
$ npm i

# Run UI
$ npm run start

# Run all tests
$ npm run test

# Run Jest Snapshot tests - *.snap.test.js
$ npm run test:snap

# Run Unit tests
$ npm run test:js
```
